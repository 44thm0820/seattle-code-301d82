Higher Order Functions

Consider the following

    
      function greaterThan(n) {
        return m => m > n;
      }
      let greaterThan10 = greaterThan(10);
      console.log(greaterThan10(11));
      // expected output → true

      // this could also be written as
      greaterThan(10)(11);
      // expected output → true

Relevant and Related Functional Programming Terms:

  • Closures
  • Currying
  • Partial Application

Open the developers console and interact with the JavaScript in app.js to learn more

The greaterThan function relies on javascript closures and is referred to as currying
